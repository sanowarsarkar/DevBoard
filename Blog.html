Question-1: What are the different ways to select an element in the DOM?

Different Ways to Select DOM Elements = Answer ( (1 = No) Select by ID: Finds an element using its unique ID. (2 = No) Select by Class Name: Selects multiple elements with the same class name. (3 = No) Select by Tag Name: Selects all elements with a specific HTML tag. (4 = No) Select by Name Attribute: Finds elements using the name attribute (mainly for form elements). (5 = No) Query Selector: Selects the first matching element using a CSS selector. (6 = No) Query Selector All: Selects all matching elements using a CSS selector. (7 = No) Loop through NodeList: Use forEach on querySelectorAll results. Convert HTMLCollection to Array: Use Array.from() on getElementsByClassName results for array methods.)

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Difference Between innerHTML, innerText, and textContent = Answer ( (1 = No) innerHTML Returns or sets an element’s HTML content. Parses and renders HTML tags. Can be a security risk if untrusted content is used. Example: Hello → Hello (bold). (2 = No) innerText Returns or sets visible text only. Ignores hidden elements and does not format HTML. Example: Hello → Hello (plain text). (3 = No) textContent Returns or sets full text content, including hidden text. Displays text without formatting. Example: Hello → Hello (plain text, no HTML parsing).)

Question-3: What is event delegation in the DOM?

Event Delegation in the DOM = Answer ( Event delegation is a technique where an event listener is added to a parent element to handle events for its child elements. Why is it needed? document.getElementById("parent").addEventListener("click", function(event) { console.log("Clicked on:", event.target); }); Here, clicking on any child element inside #parent will trigger the event. Works for dynamically added elements. Uses a single event listener instead of multiple ones. Improves memory efficiency. How does it work?)

Question-4: What is event bubbling in the DOM?

Event Bubbling in the DOM = Answer (Event bubbling occurs when a child element is clicked, and the event propagates up to its parent and ancestor Example: If a < button > is inside a < div > and the < button > is clicked, the < div > will also detect the click. How it works: If both parent and child have a click event, clicking child triggers its event first, then parent (because events bubble up).Stop bubbling: Use event.stopPropagation(); to prevent the event from propagating upward)

Question-5: How do you create, add, and remove elements using JavaScript?

Creating, Adding, and Removing Elements in JavaScript = Answer ( Creating an Element: Use document.createElement("tag") to create a new element. Example: Creating a ( p ) element → let para = document.createElement("p"); Adding an Element: Use .appendChild() or .append() to add it inside another element. Example: Adding para inside div → div.appendChild(para); Removing an Element: Use .remove() to delete an element. Example: Removing para → para.remove();)
